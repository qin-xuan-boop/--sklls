# --sklls

中文前端设计技能仓库，提供 `frontend-design` Skill。

## 包含内容

- `.agents/skills/frontend-design/SKILL.md`

## 安装到你的项目

将本仓库克隆后，把技能目录复制到你的项目根目录：

```bash
git clone https://github.com/qin-xuan-boop/--sklls.git
mkdir -p "<你的项目>/.agents/skills"
cp -R "--sklls/.agents/skills/"* "<你的项目>/.agents/skills/"
```

安装后应存在：

- `<你的项目>/.agents/skills/frontend-design/SKILL.md`

## 使用方式

在 Cursor 对话中提出前端页面/组件需求，并明确写：

- `请使用 frontend-design skill 完成该页面`

建议同时给出：

- 技术栈（React/Vue/Tailwind）
- 目标用户（例如设计团队）
- 风格方向（例如杂志感、高对比）
- 约束（深色主题、移动端、动效等）
